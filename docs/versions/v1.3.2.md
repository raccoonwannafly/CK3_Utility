# Ver 1.3.2 - Aesthetics & Data
2026-01-01

**Major Visual & Functional Update**

## Gallery Hub
*   **Sorting**: Added ability to sort characters by **Alphabetical Order** or **Date of Birth** (Chronological).
*   **Visual Enhancements**:
    *   **Immersive Fonts**: New fantasy font options (Uncial Antiqua, MedievalSharp) for headers and accents.
    *   **3D Tilt**: Character cards now react to mouse movement with a subtle 3D tilt effect.
    *   **Glassmorphism**: Premium frosted glass styling for cards and UI elements.
    *   **Animations**: Staggered fade-in animations for the character grid.
*   **Grandeur**: Fame and Devotion levels now feature tiered text styling, becoming more ornate (Gold, Glow) as the level increases.
*   **Data Completeness**:
    *   Added **Birth Name** and **Titles** fields.
    *   Historical characters updated with accurate birth dates, titles, and nicknames.
    *   These fields are now permanently visible on character cards.

## Settings
*   Added **Visual Fidelity** section to toggle animations and premium effects for performance control.
